
# WikiSim interactable - Higher Dimension

You can play it here: https://wikisim.org/wiki/1122

## Getting Started

    git clone https://github.com/AJamesPhillips/wikisim-higher-dimension
    cd wikisim-higher-dimension
    pnpm install
    pnpm run dev
